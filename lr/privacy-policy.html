Privacy Policy

Last Updated: December 26, 2024

Lightning Runner ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our GitHub Actions runner service.

1. Information We Collect

1.1 Information from GitHub

When you install and use Lightning Runner, we collect the following information through the GitHub API:

  • Repository Information: Repository names, workflow configurations, and related metadata
  • Workflow Data: GitHub Actions workflow files, job configurations, and run history
  • Account Information: Your GitHub username, organization details, and account permissions
  • Installation Data: Information about which repositories have Lightning Runner installed

1.2 Runner Execution Data

During workflow execution, we collect:

  • Job Logs: Console output and logs generated during workflow execution
  • Execution Metrics: Runtime duration, resource usage, and performance statistics
  • Build Artifacts: Temporary storage of build outputs as required by your workflows

1.3 Technical Information

  • Usage Data: API calls, feature usage, and service interaction patterns
  • Device Information: IP addresses, browser type, and operating system
  • Performance Data: Error logs, crash reports, and diagnostic information

2. How We Use Your Information

2.1 Service Provision

  • Execute GitHub Actions workflows on our infrastructure
  • Provision and manage ephemeral runner instances
  • Monitor job execution and provide real-time status updates
  • Store and retrieve build artifacts and cache data

2.2 Service Improvement

  • Analyze usage patterns to improve performance and reliability
  • Identify and fix bugs or technical issues
  • Develop new features and optimize existing functionality

2.3 Security and Compliance

  • Detect and prevent fraudulent or malicious activity
  • Enforce our terms of service and usage policies
  • Comply with legal obligations and regulatory requirements

3. Data Storage and Security

3.1 Data Storage

  • Ephemeral Execution: Runner instances are destroyed immediately after job completion
  • Artifact Retention: Build artifacts are retained according to your GitHub settings
  • Log Retention: Execution logs are retained for 90 days for debugging purposes

3.2 Security Measures

  • Encryption: All data is encrypted in transit (TLS 1.3) and at rest (AES-256)
  • Access Controls: Role-based access control and principle of least privilege
  • Network Security: Firewalls, intrusion detection, and DDoS protection
  • Monitoring: 24/7 security monitoring and incident response

3.3 Data Isolation

Each workflow execution runs in an isolated environment with no data sharing between users or organizations.

4. Data Sharing and Disclosure

4.1 Third-Party Services

We share data with the following types of service providers:

  • GitHub: To authenticate users and execute workflows
  • Cloud Infrastructure Providers: For hosting and computing resources
  • Payment Processors: For billing and subscription management (when applicable)

4.2 Legal Requirements

We may disclose your information if required to comply with applicable laws, regulations, legal processes, or valid governmental requests.

5. Your Rights and Choices

You have the following rights regarding your data:

  • Access: Request a copy of your personal data
  • Correction: Update or correct inaccurate information
  • Deletion: Request deletion of your data (subject to legal obligations)
  • Export: Receive your data in a portable format

You can manage Lightning Runner's access through GitHub Settings → Applications → Installed GitHub Apps.

6. Data Retention

We retain your data for the following periods:

  • Active Account Data: Retained while your account is active
  • Execution Logs: 90 days after job completion
  • Build Artifacts: According to your GitHub retention settings
  • Deleted Account Data: 30 days (backup retention period)

7. International Data Transfers

Lightning Runner operates globally. Your data may be processed in countries other than your country of residence. We ensure appropriate safeguards are in place for all data transfers.

8. Children's Privacy

Lightning Runner is not intended for users under the age of 13. We do not knowingly collect personal information from children.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any material changes by posting the updated policy on our website and updating the "Last Updated" date.
